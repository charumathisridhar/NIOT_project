# 🏠 3D Modeling of a Traditional Chettinad House Using Blender

A realistic 3D model of a Traditional Chettinad House created using **Blender**. This project recreates the architectural style of traditional Chettinad homes with both exterior and interior details, suitable for architectural visualization and educational purposes.

## 📌 Features

- Traditional Chettinad house design
- Front verandah (Thinnai)
- Wooden pillars and main entrance
- Sloping terracotta roof
- Interior layout with:
  - Living Hall
  - Kitchen
  - Bedrooms
  - Bathrooms
- Compound wall and entrance pathway
- Garden and surrounding environment
- Realistic materials, textures, and lighting
- High-quality rendered views

## 🛠️ Software Used

- Blender
- Python Scripting (Blender)
- AI-assisted Prompt Planning
